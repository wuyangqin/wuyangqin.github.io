Vuex

Vuex集中式存储管理应⽤的所有组件的状态，并以相应的规则保证状态以可预测的⽅式发⽣变化。

Vuex知识点回顾

核心概念

  • state 状态、数据

  • getters 派生状态

  • mutations 更改状态的函数

  • actions 异步操作

  • store 包含以上概念的容器

state 保存应用状态

mutations 修改状态

getters 从state派⽣出新状态，类似计算属性

actions 添加业务逻辑
